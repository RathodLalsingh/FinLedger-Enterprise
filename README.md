# 💰 FinLedger Enterprise

> A production-ready Personal Finance Management Backend built with **Spring Boot**, featuring secure authentication, high-performance caching, report generation, and enterprise-level architecture.

## 🚀 Features

- 🔐 JWT Authentication & Refresh Tokens
- 👤 User Registration & Login
- 💰 Income & Expense Management
- 🏷️ Category Management
- 📊 Dashboard Analytics
- 🔍 Transaction Filtering
- 🔔 Notification Management
- ☁️ Cloudinary Profile Image Upload
- 📄 Excel Report Export (Apache POI)
- 📧 Email Notifications & Daily Reminders
- ⚡ Redis Caching
- 🚀 Database Indexing & Query Optimization
- 🧪 Unit & Integration Testing
- 🛡️ Global Exception Handling & Validation
